Arcane Spell Language OFFICIAL Website

Description

Arcane Spell Language - Learn American Sign Language through a Virtual Reality Spell Battler.

This project was our final project to create a game that would help people learn American Sign Language with VR.


Hardware Used

  • Laptop
  • Meta Quest Pro
  • Meta Quest 3

    • Technologies Used

  • Unity: The software used to develop this VR Game.
  • Meta Hand Tracking SDK: This package was used for detecting hand gestures.
  • JIRA: JIRA was used to plan each sprint's stories and to assign tasks between team members. This agile platform allowed us to see how many story points needed to be closed before each deadline.

    • Project Team Members: Kriti Gupta, Derek Yang, Andy Yao
