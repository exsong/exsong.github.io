EECS 440 - CSE Simulator

Description

This project was created in Unreal 5 using UE5's Blueprint scripting. We created a creative replica of the Bob and Betty Beyster Building at the University of Michigan, with some extra functionality. You can interact with objects and NPCS, such as the copy machines, zombies, and trash. The light switches on and off and the fire alarm flashes the room's lights red. A custom interaction we added was when the player interacts with the errored out PC, robots spawn and chase the user. Will the player survive the robots and destroy them all?

Hardware Used

  • Meta Quest Pro - Using Meta Quest Link
  • Windows Desktop - GeForce RTX 4090

    • Technologies Used

  • Unreal 5: The software used to develop this VR game.
