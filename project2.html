EECS 440 - A2 Go!

Description

A2-Go! Is an educational and athletics mobile application marketed towards tourists visiting Ann Arbor. The app seeks to educate visitors on the areaâ€™s history, institutions, venues, and activity, while also getting them out and about for physical exercise and socialization.

This project was an exploration of AR techniques such as image tracking & anchoring, ar raycasting, collisions, location handling, game development, and general Unity development.


Hardware Used

  • MacBook Pro
  • iPhone (iOS)

    • Technologies Used

  • Unity: The software used to develop this AR game.
  • Mapbox: Mapbox provides location services for websites, apps, etc. We utilized it's capabilities to handle user GPS location.
  • JIRA: JIRA was used to plan each sprint's stories and to assign tasks between team members. This agile platform allowed us to see how many story points needed to be closed before each deadline.
